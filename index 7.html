Internet et Web

Comprendre le fonctionnement d'Internet et du Web.

1. Définition d’Internet : Internet est un réseau qui relie entre eux de nombreux réseaux informatiques locaux.

2. Deux types de liaisons permettant la transmission de données : • Liaisons filaires : câbles réseau, fibre optique • Liaisons hertziennes : Wi-Fi, 4G, 5G

3. Deux activités de loisirs responsables d'une grande partie du trafic Internet : • Le streaming vidéo • Les jeux vidéo

4. Appareil permettant la communication entre deux réseaux d’ordinateurs : Les routeurs

5. Qu’est-ce que le protocole IP et son rôle : Le protocole IP a pour rôle d'acheminer les données à travers le réseau, en les faisant transiter d'un émetteur vers un récepteur.

6. Qu’est-ce qu’une adresse IP ? Exemple : Une adresse IP est une adresse unique permettant au protocole IP de se repérer à travers le réseau.

7. Qu’est-ce que le protocole TCP et son rôle : Le protocole TCP assure la transmission des données lorsque deux machines communiquent. Il fixe les règles de découpage des données, d'envoi, de vérification de l'intégrité et de réassemblage.

8. Pourquoi les adresses symboliques ont-elles été créées ? Parce que les adresses IP sont difficiles à mémoriser. Les adresses symboliques sont plus faciles à retenir et rendent la navigation accessible à tous.

9. Rôle d’un serveur DNS : Le serveur DNS analyse une adresse symbolique pour trouver l’adresse IP correspondante, en sollicitant d’autres serveurs. Il agit comme un grand annuaire d’Internet.

10. Les deux grands modèles d’architecture d’échanges de données : • Client-serveur : Les ressources sont sur un serveur. Les clients envoient des requêtes pour accéder aux données. • Pair-à-pair (peer-to-peer) : Chaque machine peut être client ou serveur. Les données sont stockées sur les machines des participants.

11. Avantage et inconvénient du modèle pair-à-pair : • Avantage : Permet des téléchargements plus rapides et évite la saturation d’un seul serveur. • Inconvénient : Mauvaise réputation à cause des échanges illégaux de contenus soumis aux droits d’auteur
